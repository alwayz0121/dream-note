## ✨ Dream Note (Ver 1.0)

[꿈을 이루는 Dream Note 바로가기](https://dream-note.web.app/)<br>

각자의 목표를 향해 매일을 계획하는 여러분들의 꿈을 응원합니다. <br>

본 프로젝트는 React를 활용한 Web Application Project 입니다. <br>
글 작성, 수정, 삭제 등의 페이지로 구성되어 있습니다.<br>
Local Storage를 이용해 작성자의 글을 저장할 수 있습니다. <br>

<br>

```
Frontend | React, CSS3, Local Storage
Backend | -
```

---

<br>

## Structure

### 로딩 화면

- 로딩 화면을 구현해 Dream Note를 이용하기 전 설렘을 주었습니다.

### 메인 화면

- `localStorage`를 이용해 내용을 저장했습니다.
- 기록들을 목표에 따라, 시간 순서에 따라 정렬할 수 있습니다.
- `select` 태그에 공통되는 목표는 생략하고 나타내기 위해 `객체의 중복값 제거`를 이용했습니다.

### 쓰기, 수정하기 화면

- 두 화면은 구조가 같기 때문에 컴포넌트화 했습니다.
